# Special Love Realtime Server

A minimal Node.js + Express + Socket.IO server for instant messaging and push notification fallback.

## Features
- Real-time messaging between users (Socket.IO)
- Fallback to push notifications (OneSignal, coming next)
- CORS enabled for easy frontend integration

## Usage

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Deploy to Render, Fly.io, Railway, etc. (guide coming after push integration)
